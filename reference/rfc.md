# 10.3. RFC

RFC \[65] 真的非常低阶！这些文件说明了分配的数字丶编程 API丶及 Internet 上使用的协议。\
\
我已经在这里帮你引用了一些 RFC 的链接，所以你可以拿桶爆米花并开始你的学习之旅：\
\
RFC 1 \[66] — 第一篇 RFC；它会告诉你的事：比如什麽是 “Internet” 呢？因它融入生活，而深入了解它是如何设计的，这类的想法。［显然这份 RFC 已经过时了！］\
